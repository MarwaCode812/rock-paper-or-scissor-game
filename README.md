# Pierre, Feuille, Ciseaux 🎮

Ce projet est une implémentation du célèbre jeu **Pierre, Feuille, Ciseaux** en Python.

## Fonctionnalités

* L'utilisateur choisit entre **Pierre**, **Feuille** ou **Ciseaux**.
* L'ordinateur effectue un choix aléatoire grâce au module `random`.
* Le programme compare les deux choix et détermine le gagnant.
* Affichage du résultat : victoire, défaite ou égalité.

## Technologies utilisées

* Python 3
* Module `random`

## Objectif

Ce projet a été réalisé pour mettre en pratique les bases de Python, notamment les variables, les conditions, les boucles, les fonctions et l'utilisation du module `random`.
